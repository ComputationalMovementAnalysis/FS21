Computational Movement Analysis: Patterns and Trends in Environmental Data

Exercise 4

Learning Outcomes*

  • You are able to conceptualize a simple movement pattern and implement data structures and corresponding procedures (let’s call them algorithms…) for detecting it using R.
  • You understand the sensitivity of movement patterns to pattern parameter thresholds.

Prerequisites*

  • Readings Skills from “R for Data Science” (Wickham and Grolemund 2017): RS4.1 Chap15 Functions (19p, 269-289)
  • Readings Theory, Laube (2014): R4.1 Chap.2, p. 29-58
